# auth

## 用途
认证页面的 Maestro E2E 测试流程（登录和注册）。

## 关键文件
| 文件 | 说明 |
|------|------|
| `login.yaml` | 登录流程 — 输入凭证，验证跳转到待办列表 |
| `register.yaml` | 注册流程 — 创建新账户，验证注册成功 |

## AI 代理使用说明

### 在此目录工作
- 流程假设认证页面为初始显示的页面
- 登录流程的测试凭证必须与种子数据匹配
- 注册流程会创建新用户 — 每次运行之间可能需要清理数据
