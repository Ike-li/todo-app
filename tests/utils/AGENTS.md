# utils

## 用途
共享测试工具 — 自定义断言辅助函数和测试数据生成器。

## 关键文件
| 文件 | 说明 |
|------|------|
| `assertions.py` | 用于 API 响应验证的自定义断言函数（状态码、数据结构、错误消息） |
| `generators.py` | 测试数据生成器 — 随机邮箱、待办标题、分类名称等 |

## AI 代理使用说明

### 在此目录工作
- 使用 `generators.py` 生成唯一测试数据，避免并行测试运行之间的冲突
- 使用 `assertions.py` 辅助函数替代原始 `assert`，获得一致的错误消息
