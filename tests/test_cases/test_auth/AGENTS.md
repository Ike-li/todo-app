# test_auth

## 用途
认证端点的 API 测试 — 注册、登录和个人资料获取。

## 关键文件
| 文件 | 说明 |
|------|------|
| `test_register.py` | 注册测试 — 成功、重复邮箱、验证错误 |
| `test_login.py` | 登录测试 — 成功、密码错误、用户不存在 |
| `test_me.py` | 个人资料测试 — 使用有效令牌获取当前用户、无令牌时返回 401 |

## AI 代理使用说明

### 在此目录工作
- 测试使用 `auth_client` fixture 进行认证请求
- 注册测试通过生成器创建唯一邮箱以避免冲突
- 登录测试验证 JWT 令牌结构和过期时间
