# constants

## 用途
应用级别的常量，目前仅包含 Material Design 3 浅色和深色模式的颜色主题定义。

## 关键文件
| 文件 | 说明 |
|------|------|
| `Colors.ts` | 导出 `LightColors` 和 `DarkColors` 对象，用于 React Native Paper 主题 |

## AI 代理使用说明

### 在此目录工作
- 颜色对象在根布局（`app/_layout.tsx`）中展开到 MD3 主题
- 要添加自定义主题颜色，请在此扩展现有颜色对象
